O que é UX?

Aplicação de práticas de design centrado no usuário para gerar projetos coesos, desejáveis, com base nas experiências dos usuários.

É a percepção e resposta que resulta do uso prévio de um sistema, produto ou serviço.

From Cultura Colaborativa - Afinal, o que é UX?

Design Thinking

		É uma forma de pensar composta pelas fases: criar empatia; 
		definir o que vai ser feito; ideação; prototipação e teste.

User Experience

		É interessante pontuar que UX ou experiência do usuário 
		existe desde sempre. Desde que o humano começou utilizar 
		algum objeto para fazer alguma coisa.
